PBL HSV 顏色色卡

這份色卡整理目前 PBL 本地 OpenCV 偵測使用的顏色範圍。HSV 數值為 OpenCV 格式： H = 0-179，S = 0-255，V = 0-255。色條是近似視覺化；實際偵測以程式中的 lower/upper 為準。

A. Cup Water Level Detection

水色設定收斂為透光性的寶藍 / 深藍的延伸。 此任務目前仍以 Gemini API 為主，本地 HSV 僅作備援與 debug。

A-1 Deep / Royal Blue Water Range
lower HSV = (106, 110, 45) upper HSV = (113, 220, 155)
由 #15253f、#122340、#2b4776、#2b4575、#23324c 等深藍取色展開。

B. Detect Multimeter

三用電錶本地備援由深色機身、墨綠外殼、橘色外殼三種候選組成，再搭配矩形比例與旋鈕圓形特徵。

B-1 Dark Body Range
grayscale lower = 0 grayscale upper = 95
灰階亮度範圍，用於抓黑色 / 深灰色電錶本體。
B-2 Dark Teal Case Range
lower HSV = (86, 60, 75) upper HSV = (94, 190, 180)
由 #357b7c、#316360、#4b696a、#3a8482、#2d8281、#4e989e 等墨綠 / 藍綠取色展開。
B-3 Orange Case Range
lower HSV = (6, 120, 140) upper HSV = (14, 235, 255)
由 #ff995c、#f06f32、#fb8145、#e6743f、#ff7947 等橘色外殼取色展開。

C. Inspect The Signal Tower Light

塔燈改為正紅、正黃、正綠展開；黃色允許少量偏橘，紅色不延伸到桃紅，綠色不延伸到天藍。

C-1 Red Light Range, Low Hue
lower HSV = (0, 100, 80) upper HSV = (4, 255, 255)
正紅低 H 段。
C-2 Red Light Range, High Hue
lower HSV = (176, 100, 80) upper HSV = (179, 255, 255)
正紅高 H 補段，只補紅色環狀端點。
C-3 Yellow Light Range
lower HSV = (18, 90, 90) upper HSV = (32, 255, 255)
正黃為主，允許一點偏橘。
C-4 Green Light Range
lower HSV = (50, 80, 70) upper HSV = (75, 255, 255)
正綠範圍。

D. Detect Orange Baseball

橘色棒球改為單一橘色取樣展開，允許變髒或環境變暗造成的深橘 / 棕橘。

D-1 Dirty / Deep Orange Baseball Range
lower HSV = (6, 120, 100) upper HSV = (13, 235, 255)
由 #f27139 取色展開，後續仍搭配輪廓面積與圓度避免抓到背景。
